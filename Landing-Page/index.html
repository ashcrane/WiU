Project Name: WorldSurf

Project Description

WorldSurf.com is a web 2.0 website that is available on mobile devices. The site will have content about different surfing destinations from around the world. As a member, you will have the ability to edit information about the surf destination that you are currently at and upload that information for other members to see, plus so much more!
